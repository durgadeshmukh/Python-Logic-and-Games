# Python Logic & Application Projects
This repository contains a collection of Python-based applications focusing on core programming principles, mathematical logic, and user interaction.

## Featured Project: Automated Banking System
A terminal-based banking application that simulates real-world financial transactions.

### Key Features:
* **Deposit & Withdrawal Logic:** Real-time balance updates.
* **Input Validation:** Ensures users enter valid numerical data.
* **Modular Code:** Uses functions to handle specific banking tasks cleanly.

## Featured Project: Substitution Cipher Tool
A security-focused script that allows users to encrypt and decrypt messages using a randomized key.

### Key Features:
* **Randomized Mapping:** Generates a unique key for every session using the `random` module.
* **Full Character Support:** Handles uppercase, lowercase, numbers, and symbols.
* **User Interface:** Simple terminal prompts for seamless encryption/decryption.

## Featured Project: Python Slot Machine
A terminal-based game that simulates a high-stakes slot machine experience.
* **Dynamic Betting:** Integrated system for managing player balance and wagers.
* **Probability Logic:** Uses the `random` module to simulate symbol rows.
